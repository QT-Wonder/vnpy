# Windows环境编译

## 1. 安装工具

下载如下工具，并添加相应路径至 %PATH% 环境变量。
* [git](https://git-scm.com/download/win)
* [Python **3.7 64位**](https://www.python.org/downloads/release/python-377/) (**注意必须是Python 3.7 64位版本**)

## 2. 编译

获取代码并编译：
```
git clone https://github.com/QT-Wonder/vnpy.git
cd vnpy
install.bat
```


## 3. 脚本运行

在`vnpy`根目录运行如下命令

```
python run.py
```
